Rapidos Jeu de rapidé

Description du projet

Il s'agit d'un jeu de rapidé designé en html css, avec des fonctionnalités Javasript. Le but est de taper la lettre affiché sur l'écran. La vitesse augmente crescendo au fur et à mesure des bonnes réponses au bout de trois fautes nous sommes game-over.

Technologies utilisées dans ce projet

  • Javascript
  • HTML
  • CSS
